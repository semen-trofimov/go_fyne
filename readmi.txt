Виджет Listявляется одним из виджетов набора инструментов. Эти виджеты предназначены для создания действительно эффективных интерфейсов при представлении большого количества данных. Вы также можете увидеть Таблицу и Treeвиджеты, которые имеют похожий API. Из-за такой конструкции они немного сложнее в использовании.

Использует Listфункции обратного вызова для запроса данных, когда это необходимо. Есть 3 основных обратных вызова, Length, CreateItemи UpdateItem. Обратный вызов длины (передается первым) является самым простым, он возвращает количество элементов в данных, которые должны быть представлены. Другие относятся к шаблонам — как создаются, кэшируются и повторно используются графические элементы.

Обратный CreateItemвызов возвращает новый объект шаблона. Это будет повторно использоваться с реальными данными при представлении виджета. Размер MinSizeэтого объекта будет влиять на Listминимальный размер. Наконец UpdateItem, вызывается для применения элемента данных к кэшированному шаблону. Используйте это, чтобы подготовить содержимое к отображению.
