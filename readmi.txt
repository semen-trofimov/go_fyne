Доступны различные виджеты, чтобы предоставить пользователю возможность выбора, включая флажок, группу переключателей и всплывающее окно выбора.

Предоставляет widget.Checkпростой выбор да/нет и создается с использованием строковой метки. Каждый из этих виджетов также принимает «изменено» func(...), где параметр имеет соответствующий тип. widget.NewCheck(..)поэтому принимает stringпараметр для метки и func(bool)параметр для обработчика изменений. Вы также можете использовать это Checkedполе для получения логического значения.

Радио-виджет аналогичен, но первый параметр представляет собой фрагмент strings, представляющий каждую из опций. Функция изменения ожидает, что на stringэтот раз параметр вернет текущее выбранное значение. Вызов widget.NewRadioGroup(...) для создания виджета группы переключателей, вы можете использовать эту ссылку позже, чтобы прочитать Selectedполе вместо использования обратного вызова change.

Виджет выбора идентичен в сигнатуре конструктора виджету радио. Вместо этого при вызове widget.NewSelect(...)будет отображаться кнопка, при нажатии на которую отображается всплывающее окно, из которого пользователь может сделать выбор. Это лучше для длинных списков опций.
